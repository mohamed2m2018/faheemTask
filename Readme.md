## Installation

```sh
  git clone https://github.com/mohamed2m2018/faheemTask

  npm install or yarn

  npm start or yarn start

```

## Run on Android

```
react-native run-android
```

## Run on iOS

```
 cd ios & pod install
 cd ..
 react-native run-ios
```
